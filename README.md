# Personal Portfolio Website

森 大地 (Daichi Mori) のポートフォリオサイトのソースコードです。自身の経歴、スキル、そしてサイバーセキュリティに関する研究内容をまとめています。

## サイト構成 (Contents)

*   **index.html**: トップページ
*   **pages/background.html**: 高校時代から現在までの、部活動やアルバイトにおけるリーダーシップ経験と、そこから得た学びを記載しています。
*   **pages/research.html**: 現在取り組んでいる、POMDPベースのマルチエージェント攻撃モデリングに関する研究概要を記載しています。

## 使用技術 (Technology Used)

*   **言語:** HTML5, CSS3, JavaScript
*   **フォント:** Google Fonts (Inter, JetBrains Mono)
*   **バージョン管理:** Git / GitHub
